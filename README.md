A highly immersive, pixel-perfect web application designed for Minecraft players to instantly convert coordinates between the **Overworld** and the **Nether**. Built with a focus on authentic aesthetics and seamless user experience.

## Features

- **Instant Conversion**: Real-time calculation as you typeâ€”no "Convert" button needed.
- **Dynamic Theming**: The interface transforms completely based on the selected dimension:
  - **Overworld Mode**: Lush green/blue palette with a day-sky background.
  - **Nether Mode**: Dark red, immersive atmosphere with a custom netherrack pattern.
- **Authentic Pixel Art UI**: Custom-styled inputs, toggles, and typography using the _VT323_ font to replicate the in-game GUI feel.
- **Smart History**: Automatically saves your calculations (via the copy action) and allows you to restore them with a single click.
- **Responsive Design**: Fully optimized for desktop and second-screen usage while gaming.

## Tech Stack

- **Frontend**: React.js
- **Styling**: Tailwind CSS (v4) with heavy customization for pixel-art effects (box-shadows, gradients).
- **Build Tool**: Vite
- **Icons**: Lucide React
- **Fonts**: VT323 (Google Fonts)

## Getting Started

### Prerequisites

Ensure you have **Node.js** installed on your machine.

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/yourusername/minecraft-coordinates-calculator.git
    ```
2.  Navigate to the project directory:
    ```bash
    cd minecraft-coordinates-calculator
    ```
3.  Install dependencies:
    ```bash
    npm install
    ```
4.  Start the development server:
    ```bash
    npm run dev
    ```

## Usage

1.  Select your origin dimension using the large toggle switch.
2.  Enter your **X**, **Y**, or **Z** coordinates.
3.  The destination coordinates will appear instantly.
4.  Click **Copy Result** to save to clipboard and add the conversion to your local history.
